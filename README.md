# Sorcery

A framework which rethinks how data flows, in order to create the next generation of apps in the future.

Plays nicely with Phoenix LiveViews, or can be used alone.




## Installation


```elixir
def deps do
  [
    {:sorcery, "~> 0.3.0"},
    {:stream_data, "~> 0.6.0"},
  ]
end
```


Get started with `mix sorcery.init`
